Taches


Nous allons mettre en place trois taches.

Déplacement


Cette tache à pour but de gérer les appuis sur l'écran. Elle calcule le déplacement et envoie le mouvement dans la queue de déplacement.
Elle permet aussi de verrouiller l'écran pour pouvoir dessiner dessus. Elle s'occupera de sauvegarder le ou les dessins. Pour éviter de saturer la mémoire, on limitera la sauvegarde à 1000 points.

Joystick


Cette tache s'occupe de regarder si le joystick est utilisé. Si c'est le cas, elle envoie un message dans la queue de déplacement pour faire bouger l'image.
Comme la mesure du joystick est bruité on ajoutera une zone nul autour de l'angle 0

Affichage


Cette tache s'occupe d'afficher l'image sur l'écran. Elle regarde si un mouvement est demandé dans la queue de déplacement et si c'est le cas, elle utilise la fonction RemplirImage pour refaire les fichiers BMP à afficher.
Il faudra penser à mettre des saturations pour ne pas dépasser les limites de l'écran.
Enfin, lorsqu'il y a un mouvement, elle devra se charger de remettre le dessin qui est sauvegardé.
